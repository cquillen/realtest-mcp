ATR

Average True Range. Measures volatility as the average of true ranges over a given period.

Syntax

ATR(periods)

Parameters

periods: Integer. Number of bars to average. Common values: 10, 14, 20.

Returns

Float. The average true range value for the current bar.

Example

Strategy MyStrategy:
  ExitStop: Entry - ATR(14) * 2

Notes

ATR is commonly used for stop placement and position sizing. It adapts to changing market volatility.
