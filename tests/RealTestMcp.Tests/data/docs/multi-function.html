Highest and Lowest

These functions return the highest or lowest value of a series over a lookback period.

Highest

Syntax

Highest(series, periods)

Parameters

series: The data series to evaluate (e.g. H for high prices).

periods: Integer lookback period.

Example

// Donchian channel breakout
EntrySetup: C > Highest(H, 20)[1]

Lowest

Syntax

Lowest(series, periods)

Parameters

series: The data series to evaluate (e.g. L for low prices).

periods: Integer lookback period.

Example

// Long entry at lowest low
EntrySetup: C = Lowest(L, 10)
